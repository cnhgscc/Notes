inspect
=======

1. 简单使用
---------


1. 获取

    1.inspect.getsource : 获取源码  
    2.inspect.getmembers : 获取类/对象的所有成员函数  
    3.inspect.signature : 函数定义的参数(包含: \_\_annotations__ )  

2. 判断  
    1.inspect.isclass : 类
    inspect.isfunction : 普通函数

    inspect.ismethod: 对象的方法函数
    inspect.isbuiltin: 对象的内建函数
